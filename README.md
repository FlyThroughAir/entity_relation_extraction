# 基于依存句法分析的实体关系抽取

### 功能

使用依存句法分析抽取非结构化数据中的事实三元组（实体，关系，实体）

### 使用

* 项目需要安装HanLP，相关说明请参见[HanLP使用文档](http://hanlp.linrunsoft.com/doc.html)
* 然后运行main/Main.class即可
* 建议导入用户词典

### Dependency

HanLP依存句法分析
